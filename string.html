Escape Character

saat menuliskan string dan terdapat penggunaan kutip maka akan terjadi error atau string tidak dikenali.

untuk mengatasinya, yaitu menggunakan backslash (\).

var x = "We are the so-called \"Vikings\" from the north.";

jika dijalankan, outputnya tampil :

Atribut Output Deskripsi
\' ' kutip satu
\" " kutip dua
\\ \ garis miring terbalik
\b backspace
\n new line
\f form feed
\r carriage return
\t horizontal tabulator
\v vertical tabulator
